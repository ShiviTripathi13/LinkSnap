# LinkSnap

LinkSnap is a URL shortener that allows users to shorten their long URLs and access them via a short code.

## Features
- URL shortening
- Redirection to original URL
- Click tracking
- Basic Analytics/Stats
- Input Validation
- Link Expiration

## Security Features
- Rate limiting
- URL Validation
- Input Sanitization
